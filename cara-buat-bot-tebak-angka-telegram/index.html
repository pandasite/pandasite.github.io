Pada artikel ini, kita akan membahas cara membuat sebuah bot Telegram tebak angka dengan Python. Bot ini akan memberikan angka acak dari 1 hingga 100 kepada pengguna dan memberi mereka 10 kesempatan untuk menebak angka tersebut. Mari kita uraikan kodingan saya dan akan saya jelaskan juga setiap bagian dengan bahasa yang santai dan mudah dipahami.

Langkah 1: Persiapan

Sebelum memulai, pastikan Anda telah mempersiapkan beberapa hal berikut:

  1. Pastikan Anda sudah memiliki akun Telegram.
  2. Instal library telebot dengan perintah pip install pyTelegramBotAPI pada terminal atau command prompt.

Langkah 2: Mengimpor Library dan Mengatur Token Bot

Pertama-tama, kita akan mengimpor library yang diperlukan dan mengatur token bot yang diperoleh dari BotFather di Telegram.

import telebot
import random

bot = telebot.TeleBot('token_bot_father')

Pastikan Anda mengganti 'token_bot_father' dengan token bot asli yang diberikan oleh BotFather.

Langkah 3: Menyiapkan Variabel dan Fungsi

Kita perlu menyiapkan beberapa variabel yang diperlukan dalam permainan tebak angka. Variabel secret_number akan menyimpan angka acak yang harus ditebak oleh pengguna. max_attempts menyimpan jumlah maksimum kesempatan yang diberikan kepada pengguna, yaitu 10. attempts akan mencatat berapa kali pengguna telah mencoba menebak.

secret_number = random.randint(1, 100)
max_attempts = 10
attempts = 0

Selanjutnya, kita akan mendefinisikan beberapa fungsi bantu untuk memberikan pesan khusus kepada pengguna:

def send_sticker(chat_id, sticker_id):
    bot.send_sticker(chat_id, sticker_id)

def send_message_with_link(chat_id, message_text, url_text, url):
    markup = telebot.types.InlineKeyboardMarkup()
    markup.add(telebot.types.InlineKeyboardButton(text=url_text, url=url))
    bot.send_message(chat_id, message_text, reply_markup=markup)

Fungsi send_sticker digunakan untuk mengirim stiker kepada pengguna, dan fungsi send_message_with_link akan mengirimkan pesan dengan tautan yang dapat diklik.

Langkah 4: Menangani Perintah /start dan /help

Selanjutnya, kita akan menangani perintah /start dan /help yang dikirim oleh pengguna.

@bot.message_handler(commands=['start', 'help'])
def send_welcome(message):
    if '/help' in message.text:
        show_hint(message)
    else:
        welcome_message = "🎉 Halo Ges! Selamat datang di game Tebak Angka. Saya sudah nyiapin angka rahasia antara 1 dan 100 nih. Kamu punya 10 kesempatan buat menebak angka itu. Ayo tebak angkanya dengan ngetikkan angka tebakanmu! 💪"
        contact_text = "Kontak atau dukung coder..."
        contact_link_text = ""
        contact_url = "https://www.facebook.com/profile.php?id=100083767312898"  # Ganti dengan URL kontak atau dukungan Anda
        send_message_with_link(message.chat.id, welcome_message, contact_text, contact_url)

Jika pengguna mengirim perintah /help, bot akan memanggil fungsi show_hint untuk memberikan petunjuk. Jika pengguna mengirim perintah /start, bot akan menyapa pengguna dengan pesan selamat datang dan memberikan tautan untuk kontak atau dukungan.

Langkah 5: Menangani Perintah /stop

Kita akan menangani perintah /stop yang akan menghentikan permainan jika pengguna ingin berhenti sebelum mencapai maksimum percobaan.

@bot.message_handler(commands=['stop'])
def stop_game(message):
    bot.reply_to(message, "😢 Game berhenti deh. Kalo mau main lagi, ketik /start aja ya!")
    reset_game()

Bot akan merespons dengan pesan berhenti dan kemudian memanggil fungsi reset_game untuk mengulang permainan dengan angka acak baru.

Langkah 6: Menampilkan Petunjuk

def show_hint(message):
    global secret_number
    if 10 <= secret_number < 20:
        bot.reply_to(message, "🔎 Sipaling noob! Angka puluhan (10-19)")
    elif 20 <= secret_number < 100:
        tens_digit = secret_number // 10 * 10
        bot.reply_to(message, f"🔎 Sipaling noob! Angka puluhan (lebih besar dari {tens_digit} dan kurang dari {tens_digit + 10})")
    else:
        bot.reply_to(message, f"🔎 Sipaling noob! Angka satuan (1-9)")

Fungsi show_hint akan memberikan petunjuk kepada pengguna berdasarkan digit angka rahasia yang telah ditentukan.

Langkah 7: Menangani Setiap Pesan Lainnya (lanjutan)

@bot.message_handler(func=lambda message: True)
def handle_message(message):
    global attempts

    try:
        guess = int(message.text)
        attempts += 1

        if guess == secret_number:
            bot.reply_to(message, f"🎉 Selamat Ges! Kamu berhasil menebak angka {secret_number} dengan {attempts} percobaan. Keep it up! 💪")
            send_sticker(message.chat.id, 'CAACAgIAAxkBAAHR-TBkumuUBmVOE2jl_z2Ji2dKYt5XdwACswsAAipQUUoso7YJ7GnT1i8E')
            bot.reply_to(message, "🔄 Ketik /start untuk main lagi ya Ges!")
            reset_game()
        elif attempts >= max_attempts:
            bot.reply_to(message, f"😔 Kesempatan kamu sudah habis nih Bro, noob sih!  Angka yang bener adalah {secret_number}. Semoga lebih beruntung di kesempatan berikutnya! 🍀")
            send_sticker(message.chat.id, 'CAACAgIAAxkBAAHR-Udkumv3cnhJUpaVcBQkXKKRY0ezFgACZAADwZxgDM3er0WehrEhLwQ')
            bot.reply_to(message, "🔄 Ketik /start untuk main lagi ya Bro!")
            reset_game()
        elif guess < secret_number:
            bot.reply_to(message, "📉 Angka tebakan kamu terlalu kecil. Coba lagi yaa!")
        else:
            bot.reply_to(message, "📈 Angka tebakan kamu terlalu besar. Coba lagi yaa!")
    except ValueError:
        bot.reply_to(message, "❌ Ups! Mohon kirimkan angka yang valid ya.")

Pada langkah ini, kita menggunakan dekorator @bot.message_handler(func=lambda message: True) untuk menangani setiap pesan yang dikirim oleh pengguna selain perintah khusus.

Kita mencoba mengonversi pesan pengguna menjadi tipe data integer dengan guess = int(message.text). Jika pengguna mengirim teks yang bukan angka, kita akan menangkap ValueError dan memberi tahu pengguna untuk mengirimkan angka yang valid.

Selanjutnya, kita memeriksa apakah tebakan pengguna tepat dengan angka rahasia yang ada. Jika benar, kita memberikan selamat kepada pengguna dan memberikan opsi untuk memulai permainan baru dengan perintah /start. Jika pengguna telah mencapai batas maksimum percobaan (attempts >= max_attempts), kita memberitahukan angka yang benar dan memberikan opsi untuk memulai permainan baru.

Jika tebakan pengguna belum benar, kita akan memberikan petunjuk apakah angka tebakan terlalu besar atau terlalu kecil dan meminta pengguna untuk mencoba lagi.

Langkah 8: Fungsi reset_game

def reset_game():
    global secret_number, attempts
    secret_number = random.randint(1, 100)
    attempts = 0

Fungsi reset_game akan mengatur ulang nilai secret_number dengan angka acak baru antara 1 dan 100, serta mengatur ulang nilai attempts menjadi 0 untuk memulai permainan baru.

Langkah 9: Memulai Bot Telegram

bot.polling()

Untuk menjalankan bot Telegram, kita menggunakan bot.polling(). Fungsi ini akan memulai bot dan menjaga koneksi dengan Telegram agar dapat menerima dan menangani pesan dari pengguna.
