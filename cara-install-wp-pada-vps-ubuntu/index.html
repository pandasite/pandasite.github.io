Cara Install Wordpress CMS Pada Vps Ubuntu

Selamat datang para orang keren! Jika kamu ingin menghosting website atau blog pribadi dengan kontrol penuh dan kebebasan kreatif, menginstal Web Server Apache dan WordPress di Linux Ubuntu adalah pilihan yang tepat. Dengan Apache, server web yang populer dan andal, serta WordPress, platform manajemen konten yang mudah digunakan dan dapat disesuaikan, kamu bisa membuat website sesuai dengan keinginanmu. Mari kita lihat proses lengkapnya!

Apa itu Apache dan WordPress?

  • Apache: Apache HTTP Server, yang biasa dikenal sebagai Apache, adalah web server open-source yang sangat populer. Ini berarti, server web ini bersifat gratis dan siapa saja dapat menggunakannya. Apache telah menjadi salah satu web server yang paling banyak digunakan di seluruh dunia karena kinerjanya yang stabil dan fleksibilitasnya yang tinggi.

  • WordPress: WordPress adalah platform manajemen konten (CMS) open-source yang sangat populer, yang memungkinkan pengguna untuk membuat dan mengelola website dengan mudah tanpa harus menguasai bahasa pemrograman. Dengan berbagai tema dan plugin yang tersedia, WordPress menyediakan banyak opsi untuk membuat website yang unik dan sesuai dengan kebutuhan pengguna.

Persiapan sebelum Memulai

Sebelum kita masuk ke instalasi, pastikan kamu memiliki:

  1. Sistem Ubuntu Terupdate: Pastikan sistem Ubuntu kamu sudah terupdate dengan perintah sudo apt update dan sudo apt upgrade.

  2. Akses sebagai Superuser: Pastikan kamu memiliki akses sebagai superuser (atau dengan perintah sudo) untuk menginstal paket dan mengkonfigurasi server.

Langkah 1: Instal Apache

Pertama, mari kita instal Apache Web Server. Buka terminal dan ketikkan perintah berikut:

sudo apt install apache2

Setelah instalasi selesai, Apache akan secara otomatis mulai berjalan. Kamu bisa memeriksa statusnya dengan menggunakan perintah:

sudo systemctl status apache2

Jika berhasil, kamu akan melihat status “active (running)” yang menandakan Apache berjalan dengan baik.

Langkah 2: Instal MySQL

WordPress membutuhkan basis data untuk menyimpan konten dan pengaturan. Untuk itu, kita perlu menginstal MySQL dengan perintah berikut:

sudo apt install mysql-server

Selama instalasi, sistem akan meminta kamu untuk mengatur password untuk pengguna root MySQL. Pastikan kamu menggunakan password yang kuat dan ingat dengan baik.

Langkah 3: Instal PHP

Kita juga perlu menginstal PHP dan beberapa ekstensi yang dibutuhkan oleh WordPress:

sudo apt install php libapache2-mod-php php-mysql php-curl php-gd php-mbstring php-xml php-xmlrpc

Langkah 4: Konfigurasi PHP

Buka file konfigurasi PHP untuk mengatur beberapa pengaturan yang diperlukan oleh WordPress:

sudo nano /etc/php/7.x/apache2/php.ini

Ubah nilai beberapa variabel sesuai dengan yang dijelaskan pada Langkah 6 di panduan sebelumnya.

Langkah 5: Restart Apache

Setelah mengubah konfigurasi PHP, restart Apache untuk menerapkan perubahan:

sudo systemctl restart apache2

Langkah 6: Instal WordPress

Sekarang, saatnya menginstal WordPress CMS. Unduh paket WordPress dari situs resminya:

cd /tmp
wget https://wordpress.org/latest.tar.gz

Ekstrak paket yang telah diunduh:

tar -xvzf latest.tar.gz

Pindahkan isi folder WordPress ke direktori root Apache (/var/www/html):

sudo rsync -avP /tmp/wordpress/ /var/www/html/

Langkah 7: Konfigurasi WordPress

Buat database dan user MySQL baru untuk WordPress:

mysql -u root -p

Setelah masuk ke MySQL, buat database baru:

CREATE DATABASE nama_database;

Buat user baru:

CREATE USER 'nama_user'@'localhost' IDENTIFIED BY 'password_user';

Beri izin user untuk mengakses database:

GRANT ALL ON nama_database.* TO 'nama_user'@'localhost';
FLUSH PRIVILEGES;
EXIT;

Ubah nama file wp-config-sample.php menjadi wp-config.php:

sudo mv /var/www/html/wp-config-sample.php /var/www/html/wp-config.php

Buka file wp-config.php dengan menggunakan editor teks:

sudo nano /var/www/html/wp-config.php

Ubah nilai dari 'database_name_here', 'username_here', dan 'password_here' sesuai dengan informasi database dan user MySQL yang telah kamu buat sebelumnya.

Langkah 8: Selesaikan Instalasi WordPress melalui Browser

Buka browser dan masukkan alamat IP server Ubuntu atau nama domain (jika sudah ada). Kamu akan diarahkan ke proses instalasi WordPress. Isi informasi yang diminta seperti judul website, nama pengguna, dan password untuk akun admin WordPress.

Selamat, kamu berhasil menginstal WordPress! Kamu sekarang dapat masuk ke dashboard WordPress dan mulai berkreasi dengan website baru kamu.

Akhir Kata

Menginstal Apache Web Server dan WordPress pada Linux Ubuntu memungkinkan kamu untuk meng-host website dengan mudah dan fleksibel. Kamu bisa menyesuaikan website sesuai dengan kebutuhanmu menggunakan tema dan plugin yang disediakan oleh WordPress. Selamat mencoba dan semoga website baru kamu sukses dan bermanfaat! 🌟
